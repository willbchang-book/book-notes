## 方法
- **使用方法**：在电脑上使用 [Emacs](https://www.gnu.org/software/emacs/) 的 [Org Mode](https://orgmode.org/) 进行编辑、阅读和学习，配置为 [Ward Emacs](https://github.com/willbchang/ward-emacs)。在不方便使用电脑时，通过 iCloud 同步，在手机上使用 [orgro](https://github.com/amake/orgro) 进行阅读和复习。
- **笔记方法**：结合[康奈尔笔记法](http://lsc.cornell.edu/how-to-study/taking-notes/cornell-note-taking-system/)、 [SQ4R 反思式阅读法](psychology/心理学导论.org#反思阅读法-SQ4R)进行笔记和学习。因为 Org Mode 可以自由地折叠和展开层级，所以可以把各个层级的标题当作关键词使用，梳理出大纲。同时也可以在每个标题下记录关键问题，便于自测。另外 Org Mode 也可以分别统计每章或每小节的阅读时间及其总和。
